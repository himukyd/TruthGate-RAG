Answer:
To define a path parameter in FastAPI, you use the `Path` decorator. For example:
```python
from fastapi import Path

@app.get("/items/{item_id}")
def read_item(item_id: int = Path(...)):
    return {"item_id": item_id}
```
This sets `item_id` as a path parameter that must be provided in the URL. The `Path(...)` annotation specifies that the parameter is required. The `...` indicates that the parameter is optional, but it is recommended to always provide a value for path parameters.
