Question: How do I define a path parameter in FastAPI? 

Answer:

To define a path parameter in FastAPI, you can use the `Path` decorator on your route function. For example:

```python
from fastapi import FastAPI, Path

app = FastAPI()

@app.get("/items/{item_id}")
def read_item(item_id: int = Path(...)):
    return {"item_id": item_id}
```
